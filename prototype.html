What is CyberScan?

CyberScan is a local Python-based web vulnerability scanner for authorized target websites. It helps students demonstrate safe scanning, findings review, and report generation.

What is a vulnerability scan?

A vulnerability scan checks a website for common security weaknesses such as missing headers, insecure cookies, exposed paths, and safe SQL or XSS indicators.

What is OWASP?

OWASP is a widely used application security community and framework. CyberScan maps findings to common OWASP-style web risk categories.

Why is authorization required?

Scanning systems without permission can be disruptive or illegal. CyberScan is intended only for websites you own or have written permission to test.

What do risk levels mean?

Critical and High findings need urgent review, Medium findings should be planned for remediation, Low findings improve hardening, and Info findings provide context.
